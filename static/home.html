Roblox Proxy and APIs

Quick About

This is an open-source roblox proxy for accessing roblox.com from in-game with HttpService, it also hosts some custom APIs.

Usage

Just like roproxy, simply enter the roblox link you want to go to with roblox.com replaced to this domain. For example, if you wanted to get user information by username: https://api.rprxy.xyz/users/get-by-username?username=Froast. Subdomain support is managed manually: if there's a subdomain that isn't available you want me to add just let me know.
